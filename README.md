# kjj403.github.io

간단한 포트폴리오 정적 사이트입니다. GitHub Pages로 자동 배포되도록 워크플로우가 포함되어 있습니다.

로컬에서 확인하기:

```bash
# 로컬 서빙
python3 -m http.server 8000
# 브라우저에서 열기: http://localhost:8000
```

배포 방법: 변경사항을 `main` 브랜치로 푸시하면 `.github/workflows/deploy.yml`이 GitHub Pages로 자동 배포합니다.

기술 스택: HTML, CSS, JavaScript

원하시면 디자인, 프로젝트 설명, 실제 프로젝트 링크 등을 더 정교하게 바꿔 드립니다.
